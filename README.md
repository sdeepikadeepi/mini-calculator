# Calculator Web App

## 📌 Description
A simple and responsive calculator web application built using HTML, CSS, and JavaScript. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division with a clean user interface.

## 🚀 Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Clear button
- Responsive design
- Easy to use interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Files
- index.html
- style.css
- script.js

## 🎯 Purpose
This project is designed to perform quick mathematical calculations in a simple and user-friendly way.

## 📸 Output
Calculator interface with buttons and display screen.
